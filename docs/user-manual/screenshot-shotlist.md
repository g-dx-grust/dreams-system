# 画面キャプチャ撮影リスト

利用説明書に差し込む画面を、撮影順と役割ごとに整理したリストです。まずは必須画面だけを揃え、その後に補助画面を追加してください。

自動撮影を使う場合は `docs/user-manual/assets/README.md` のコマンドを実行してください。

## 撮影ルール

- 画面幅は原則デスクトップ表示で統一する
- ブラウザ倍率は `100%` に固定する
- 氏名、住所、電話番号、メールアドレスなどの個人情報は必ずダミーデータにする
- 締切超過、未収などを説明したい画面は、状態が分かるサンプルデータを用意する
- 可能なら同じ案件を使って一連の画面を撮影し、説明の流れを揃える

## ファイル命名規則

`{連番}-{画面名}.png`

例:

- `01-login.png`
- `05-case-create.png`
- `10-case-documents.png`

保存先:

`docs/user-manual/assets/`

## 補助ショット

補助ショットは、説明書を読みやすくしたいときに追加します。

| 画面 | ルート | 用途 |
|---|---|---|
| 人物登録 | `/persons/new` | 人マスタ登録の詳細説明 |
