# 案件管理・帳票転記システム 利用説明書

この文書は、社内利用者向けの操作説明書です。本日の開発状況説明でも使えるよう、主要画面のキャプチャを差し込んだ状態にしています。

## 1. 本システムでできること

本システムは、測量・許認可業務に関する情報を案件単位で管理し、登録済みデータを各種帳票へ転記するための業務システムです。

- 案件ごとの進捗、提出先、期限、担当者を管理できます
- 人マスタを再利用して、案件関係者の重複入力を減らせます
- 土地情報と金額情報を案件に紐づけて管理できます
- テンプレートを選んで Word / Excel 帳票を生成できます
- 生成履歴と監査ログから操作履歴を追跡できます

## 4. 基本的な業務の流れ

### 4-1. 人マスタを登録する

案件に紐づける前に、申請者や所有者などの人物情報を人マスタへ登録します。

主な登録項目:

- 個人 / 法人の区分
- 氏名、フリガナ
- 住所
- 電話番号などの連絡先

画面イメージ:

![人マスタ一覧](./assets/11-persons-list.png)

### 4-2. 案件を登録する

案件画面から新規案件を登録します。案件名、案件種別、担当者、提出先、提出日、締切日などを入力します。

案件登録画面:

![案件登録画面](./assets/04-case-create.png)

登録後は案件ごとの基本情報画面で、ステータスや担当者、提出先、期限などを更新します。

案件基本情報画面:

![案件基本情報画面](./assets/05-case-basic.png)

### 4-3. 案件に関係者を追加する

案件詳細の `関係者` タブでは、人マスタから人物を選んで案件へ追加できます。追加時点の情報はスナップショットとして案件側に保持されます。

利用者向けの注意点:

- 人マスタの値を後から変更しても、案件に保存済みのスナップショットは自動では変わりません
- 最新のマスタ値を反映したい場合は `マスタから再同期` を使います

画面イメージ:

![案件関係者タブ](./assets/06-case-persons.png)

### 4-4. 土地情報を登録する

`土地情報` タブでは、筆ごとの所在地、地番、地目、地積、転用面積を登録します。複数筆がある案件では、筆を追加してまとめて管理できます。

画面イメージ:

![土地情報タブ](./assets/07-case-parcels.png)

### 4-5. 金額情報を登録する

`金額` タブでは、見積金額、請求金額、入金金額、税率、未収金を確認・管理できます。

画面イメージ:

![金額タブ](./assets/08-case-financial.png)

### 4-6. 帳票を生成する

`帳票生成` タブでは、対象テンプレートを選択して、帳票生成前チェックと帳票出力を行います。

運用上のポイント:

- 生成前に `転記前チェック` を実施します
- 必須フィールドが不足している場合、帳票生成は行わず先に案件データを補完します
- 必要に応じて転記箇所のハイライトを有効にして出力します

画面イメージ:

![帳票生成タブ](./assets/09-case-documents.png)

### 4-7. 生成履歴を確認する

案件ごとの `生成履歴` タブ、または全体の `帳票履歴` 画面から、生成済み帳票の確認とダウンロードができます。

案件単位の生成履歴:

![案件生成履歴](./assets/10-case-history.png)

システム全体の帳票履歴:

![帳票履歴画面](./assets/12-documents-history.png)

## 7. よくある注意点

- 案件に追加した関係者情報は、その時点のスナップショットとして保存されます
- 帳票生成前チェックで必須項目不足が出た場合は、案件情報または人マスタを先に補完してください
- 管理者専用メニューは、一般ユーザーには表示されません
- 監査ログには主要な操作履歴が記録されます
