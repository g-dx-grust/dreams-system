# ClaudeCodeルール — グラスト共通開発ルール

このディレクトリは、株式会社グラスト（GRUST.inc）が開発するすべての業務システムに
横断適用される **Claude Code 用の共通ルール** を保管する場所です。

## 収録ファイル

| ファイル | 役割 |
|---|---|
| `CLAUDE.md` | グラスト共通の最上位ルールブック（最優先） |
| `G-DX_Design_System.md` | デザイントークン／コンポーネント規定／アクセシビリティの**正本** |

## 各システムからの参照方法

各リポジトリの `CLAUDE.md` 冒頭に、以下のように本ディレクトリを参照する記述を入れてください。

```markdown
## 参照ドキュメント
- グラスト共通ルール: `../ClaudeCodeルール/CLAUDE.md`（最優先）
- デザインシステム: `../ClaudeCodeルール/G-DX_Design_System.md`（正本）
- 本システム要件: `./docs/...`
```

各システム固有のルール（要件定義の参照先・ロール定義・リリース順など）は
各リポジトリの `CLAUDE.md` で本共通ルールを上書き／追記してください。
**本共通ルールと矛盾するシステム固有ルールは書かないでください。**

## 編集ルール

- `G-DX_Design_System.md` の重複コピーを各リポジトリに置かない（ここが唯一の正本）
- トークン追加・変更時は `G-DX_Design_System.md` の変更履歴セクションに追記
- 共通ルール改定時は `CLAUDE.md` の変更履歴セクションに追記
- 変更は必ずユーザー承認後に行う
